INPUT: A markdown network automation report

OUTPUT FORMAT (when risks, considerations or recommendations exist):
{
  "create_issue": true,
  "title": "<concise issue title>",
  "priority": "<💀critical|🔥high|⚠️medium>",
  "description": "<detailed markdown description including key recommendations. Append a task list with check boxes to fulfill those recommendations>"
}

RULES:
- Title: Max 80 chars, actionable (e.g., "Implement BGP route filtering on core routers")
- Priority: Based on risk/impact mentioned in report
- Description: Include context from Executive Summary + specific recommendations
- No meta-text, only valid JSON
